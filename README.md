# 🗂️ Lee — 好用项目收藏

收集别人做的好项目，方便以后查阅和使用。

---

## 项目清单

### 1. markitdown (Microsoft)
- **用途：** 文档转 Markdown。支持 Word、PPT、Excel、PDF、HTML 等格式
- **适用：** 所有 AI Agent
- **链接：** [github.com/microsoft/markitdown](https://github.com/microsoft/markitdown)

### 2. headroom
- **用途：** 压缩工具输出和日志，减少 60%-95% 的 Token 消耗，答案质量不变
- **适用：** 支持 MCP 的 AI Agent
- **链接：** [github.com/chopratejas/headroom](https://github.com/chopratejas/headroom)

### 3. ECC
- **用途：** 把仓库历史变成可复用的工作流、技能、规则和安全审计，优化 AI 编程 Agent
- **适用：** Claude Code / OpenCode
- **链接：** [ECC Tools 组织页](https://github.com/ECC-Tools)

### 4. codegraph
- **用途：** 本地运行的代码知识图谱，帮助 AI 理解整个代码库，省 Token 省调用
- **适用：** Hermes / Claude Code / Codex / Gemini / Cursor / OpenCode 等
- **链接：** [github.com/colbymchenry/codegraph](https://github.com/colbymchenry/codegraph)

### 5. Understand Anything ⭐54.9K
- **用途：** 把任意代码库、知识库转成交互式知识图谱，可探索、搜索、提问
- **适用：** Claude Code / Codex / Cursor / Copilot / Gemini CLI 等
- **链接：** [github.com/Lum1104/Understand-Anything](https://github.com/Lum1104/Understand-Anything)

### 6. taste-skill ⭐38.0K
- **用途：** 提升 AI 的审美和输出质量，避免生成枯燥、同质化的内容
- **适用：** 所有 AI Agent
- **链接：** [github.com/Leonxlnx/taste-skill](https://github.com/Leonxlnx/taste-skill)

### 7. supermemory ⭐26.1K
- **用途：** AI 持久记忆引擎，跨会话保存上下文，解决 Agent 遗忘问题
- **适用：** 所有 AI Agent（通过 Memory API）
- **链接：** [github.com/supermemoryai/supermemory](https://github.com/supermemoryai/supermemory)
